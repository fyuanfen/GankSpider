

利用Gank的API，爬取Gank干货集中营的将Gank中的资源存储到leancloud上


其中Leancloud1.0.py按照**版块**存储旧版的无图片的数据,即2016/06/25改版之前的数据


Leancloud2.0按照**日期**存储新版干货集中营的数据



---



## Step1：Run "leancloud1.0.py" to store the old edition of GankIO 

## Step2：Run the function"createPage()" in leancloud2.0.py to store the latest edition of GankIO so far


### Attention: you can only run the function "createPage()"once
if you want to update your date,please run the function "updatePage()",you will be appreciated to run it everyday ###
